###“指令:otanow - 立即执行OTA升级”使用说明
####指令的作用
　　使用 otanow 指令，可以让设备执行一次在线自动升级。
　　使用时注意以下事项：
　　(1) 发出 otanow 指令后，开发板在**90秒内不能断电**，否则会导致升级失败，无法开机。
　　(2) 开发板升级完成后，会自动重启。您可以登录开发板后台的首页面，查看当前固件版本号。
　　(3) 开发板已经连入的WIFI热点，必须能正常访问互联网。

####指令的格式（示例）
```
http://192.168.7.170:38585/ctrl?token=3f4bffa77257d243875d0a5a80635934&amp;cmd=otanow
```
####指令的格式（TCP方式示例）
```
{&quot;cmd&quot;:&quot;otanow&quot;, &quot;tid&quot;:&quot;1234&quot;}\x11\x12
```
####指令参数说明
 | 参数名  | 参数值（示例）  |参数类型|说明|
| ------------ |:------------ |-----|--|
|token |3f4bffa77257d243875d0a5a80635934 |字符型|管理员Token|
|cmd  | otanow |字符型|控制指令名称|
|tid (可选)  | 任意值 |字符型|仅TCP接口方式可用。如省略，则指令执行后，不反馈指令执行结果。|
####指令反馈结果
成功时返回：
```
{
  &quot;code&quot;: 0
}
```

失败时返回：
```
{
  &quot;code&quot;: 101,
  &quot;msg&quot;: &quot;Invalid arguments.&quot;
}
```

####指令反馈结果说明
无
